# FrontEndChallenge
 
A simple calendar app.

<h2>Feactures</h2>
<ul>
 <li>
-Add, delete and edit events (Reminders)
 </li>
  <li>
-Custom color for each reminder
 </li>
  <li>
-Weather  based on the city and reminder's date
 </li>

 </ul>


<h2>Dependences</h2>
<ul>
 <li>
-@syncfusion/ej2-react-schedule v 17.4.51
 </li>
  <li>
-Redux v 4.0.5
 </li>
 <li>

-Redux-react v 7.2.0
 </li>
 </ul>
